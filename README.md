# 📊 Sales Dashboard (Power BI Project)

## 🚀 Project Overview

This project presents an interactive **Sales Dashboard** built using **Power BI**.
It helps analyze sales performance, revenue trends, and customer insights using real-world dataset (CSV files).

---

## 📁 Project Structure

```
Sales-Dashboard/
│── data/               # CSV files used for analysis
│── dashboard/          # Power BI (.pbix) file
│── images/             # Dashboard screenshots
│── README.md
```

---

## 📌 Features

* 📈 Sales trend analysis (monthly/yearly)
* 🛍️ Product performance insights
* 🌍 Region-wise sales distribution
* 👥 Customer segmentation
* 💰 Revenue and profit tracking
